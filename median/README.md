## Dependencies

* [rust](https://rustup.rs/)
* [cargo make](https://github.com/sagiegurari/cargo-make)
  * `cargo install cargo-make`

## Examples

checkout the examples in **median/examples**
You can build/package/install them with:

`cargo make build`
`cargo make package`
`cargo make install`
